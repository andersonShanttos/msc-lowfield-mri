# Projeto de Mestrado – Ressonância Magnética de Baixo Campo

Este repositório contém materiais de pesquisa relacionados a um projeto
de mestrado focado no desenvolvimento e validação de um sistema de
ressonância magnética (RM) de baixo campo, em ambiente de laboratório.

## Escopo do projeto
- Desenvolvimento em laboratório (sem aplicação clínica)
- Ressonância Magnética de baixo campo (low-field MRI)
- Componentes de hardware e software
- Validação utilizando phantoms
- Ênfase em reprodutibilidade e documentação técnica

## Status
- Projeto em fase inicial
- Estrutura e conteúdos sujeitos a alterações

## Observação
Este repositório é de uso acadêmico e está configurado como privado.
